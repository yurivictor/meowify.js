# Meowify.js
Changes a string into meows.

_What is it good for? Absolutely nothing._

## How to use

```
<script src="meowify.min.js"></script>
<script>
var example = meowify( 'I am an example string.' );
console.log( example ); // Mw mw mw meeeeeew meeeeeow.
</script>
```
